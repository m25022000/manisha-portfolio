# Artifact 2 – AI/ML Leadership and Change Management

Created by **Manisha Katamoni**

This updated version includes an interactive AI Change Readiness Lab and Leadership Decision Simulator.

## Publish on GitHub Pages
1. Create a repository named `artifact-2-ai-leadership`.
2. Upload `index.html` and `README.md`.
3. Open **Settings > Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/ (root)`.
6. Save and open the generated public URL.

No dependencies or installation are required.
